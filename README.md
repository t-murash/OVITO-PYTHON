## ovito-python の準備

事前に Miniconda3 をいれておく。

- [Miniconda3](https://docs.conda.io/en/latest/miniconda.html)

```
$ conda create -n ovito
$ conda activate ovito
```

ovito を使う場合は毎回 `conda activate ovito` を実行する。
